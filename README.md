## Overview

**Depth Oracle** is a full-stack, AI-powered application rooted in Jungian psychology, designed to guide users toward profound personal growth and emotional clarity. Built with Next.js and hosted on Vercel, it offers an intuitive platform for exploring the unconscious through conversations with Elara, an AI companion. Elara provides empathetic, archetype-driven insights, drawing from Jungian concepts like the shadow, anima/animus, persona, and individuation. With tools such as active imagination, dream amplification, and reflective journaling, Depth Oracle transforms inner exploration into actionable steps for lasting self-integration.

**Tagline**: "Unveil Your Depths: Elara Illuminates the Path to Wholeness and Insight."
